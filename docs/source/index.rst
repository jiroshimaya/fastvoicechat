FastVoiceChat ドキュメント
===================================

バージョン: |version|

FastVoiceChat は、Pythonで実装された高速で使いやすい音声チャットライブラリです。
リアルタイムの音声処理と自然な音声合成を組み合わせることで、
インタラクティブな音声コミュニケーションを実現します。
